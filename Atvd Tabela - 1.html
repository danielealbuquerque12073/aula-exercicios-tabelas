Descrição das TAGS

  • TABLE: Representa uma tabela em HTML

  • TH: Representa o cabeçalho da tabela

  • TR: Representa cada linha da tabela

  • TD: Representa cada célula de uma tabela

  • THEAD: Representa o cabelhaço da tabela

  • TBODY: Representa o corpo da tabela

  • TFOOT: Representa o rodapé da tabela

  • ALIGN: Atributo usado para alinhar elementos numa página HTML

  • BORDER: Utilizado para inserir bordas na tabela

  • CAPTION: Representa a legenda da tabela

  • ROWSPAN: Usado para mesclar linhas de uma tabela

  • COLSPAN: Usado para mesclar colunas de uma tabela

  • WIDTH: Especifica uma largura para cada célula de uma tabela
